## 🚀 التثبيت والتشغيل

### 1. تثبيت المتطلبات
```bash
pip install -r requirements.txt
```

### 2. إعداد مفتاح Groq API
أنشئ الملف: `.streamlit/secrets.toml`
```toml
GROQ_API_KEY = "gsk_your_key_here"
```
احصل على مفتاحك المجاني من: https://console.groq.com

### 3. تشغيل التطبيق
```bash
streamlit run app.py
```
